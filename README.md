## Snake AI Game
A simple AI Reinforcement Learning project to control ping pong game

- ### Train and Run Ai
  - Run ./ai/train_ai.py

- ### Play the Game
  - Run ./game/run.py

## Requirements:
- python==3.7
- tensorflow==2.6.0
- pygame

## Tweaking the AI
You can tweak the AI constants to try out or better the AI training, goto ./ai/constants.py
